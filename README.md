# Indian Cement Valuation Terminal

An interactive equity research dashboard built using the Damodaran FCFF valuation framework to analyze leading Indian cement companies.

## 📊 Overview

This project analyzes four major cement companies:

- UltraTech Cement  
- Shree Cements  
- Dalmia Bharat  
- Ramco Cements  

The terminal evaluates:

- Revenue growth trends  
- EBIT margins  
- ROIC vs WACC (value creation test)  
- Sales-to-capital efficiency  
- Free Cash Flow projections  
- Reverse DCF assumptions  

---

## 🧠 Core Insight

"Cement is a capital discipline game disguised as a growth story."

Growth alone does not create value — only when ROIC exceeds WACC does real value emerge.

## 📈 Valuation Framework

- FCFF (Free Cash Flow to Firm) model  
- Discounted Cash Flow (DCF) valuation  
- Reverse DCF analysis  
- Terminal value sensitivity  

Based on concepts from Aswath Damodaran.

---

## 🔍 Key Takeaways

- UltraTech: Scale advantage, moderate upside  
- Shree: Best capital efficiency, strongest balance sheet  
- Dalmia: Overpriced narrative-driven growth  
- Ramco: High leverage, delayed value creation
